# Lunar Judgement 

This is a game built in Python with Pygame which features Eldermoon as protagonist and decends to the land when the vessel of the sin overflows to remove those from the existance who walks the path of *ADHARMA*.

This game features:<br>
* Good Animation
* Sound Effects
* Challenging Boss Battle

Demo:<br>
![Gameplay](./assets/demo.mp4)

### Prerequisites
---
 - Installed Python (with version that supports pygame-ce)
 - Installed Pygame-ce

### Installation
---
 ```
 git clone https://github.com/BrishkamalKarki/lunar-judgement-game
 cd lunar-judgement-game
 ```
 Install Dependencies:
 ```
 pip install pygame-ce
 ```
 Run the Game:
 ```
 python game.py
 ```
